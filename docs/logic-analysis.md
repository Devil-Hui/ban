## 十二、已确认决策汇总（v3.4 全部闭合）

| # | 原问题 | 最终决策 |
|---|--------|---------|
| 1 | 后端方案 | CloudBase Serverless + MySQL |
| 2 | 数据存储 | CloudBase MySQL + 云存储 |
| 3 | 组件库 | TDesign WeChat 组件库 |
| 4 | 底部导航 | 4 Tab：首页/日程/任务/个人中心 |
| 5 | 个人中心 | 6 大板块（分组/历史/日历/推送/设置/关于） |
| 6 | 联系人详情 | 发布者可见脱敏手机号；成员间仅姓名+时段 |
| 7 | 排班周范围 | 周一~周日 默认 + 单双周 Switch + 自定义 |
| 8 | 上传方式 | 仅拍照+相册；强制 imgSecCheck |
| 9 | 通知方式 | 4 模板订阅消息（创建/截止/查收/更新） |
| 10 | MVP 范围 | AI 用 OCR API；排班服务端随机抽取 |
| 11 | 场景 | 校园→通用轻量协同排班平台 |
| 12 | 权限模型 | Superadmin/Admin(H5) + Publisher/Joiner(小程序) |
| 13 | AI 策略 | 腾讯云 OCR API + 简单规则，无本地模型 |
